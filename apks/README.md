# Dr. Clicker — APK Downloads

APK files in this folder are auto-generated by GitHub Actions on every push to `main`.

| File | Description |
|------|-------------|
| `DrClicker-debug.apk` | Debug build — use this for testing |
| `DrClicker-release-unsigned.apk` | Release build (unsigned) |

> **Latest release with download links →** see the [Releases](../../releases) tab.

## Install Instructions

1. On your Android device (Android 11+): **Settings → Security → Unknown sources → ON**
2. Transfer the APK (USB / Google Drive / WhatsApp)
3. Open the APK file to install
4. Grant **Accessibility** and **Draw over other apps** permissions when prompted
5. Load the yellow Accept button crop as a template
6. Toggle **Engine Activation** ON
